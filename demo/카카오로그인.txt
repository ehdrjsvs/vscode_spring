작업지시서

1.웹서버 주소 : http://localhost: 8000

2.클라이언트키 : 896f11cd1644ef2adbd2eff3c8f7fe3a //rest api key

3.카카오 로그인 요청 콜백 주소
http://localhost:8000/auth/kakao/callback   //Redirect URI

4.카카오 로그아웃 콜백주소
